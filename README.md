# Uber Ride Analytics Dashboard

## Project Overview

This project analyzes Uber ride booking data using Power BI and provides insights into bookings, revenue, vehicle performance, rider behavior, and cancellation trends.

## Tools Used

* Power BI
* DAX
* Power Query
* Excel

## Dashboard Pages

### Home Page

* Project navigation
* Interactive vehicle selection

### Overview Page

* Completed Bookings
* Lost Bookings
* Revenue
* Total Distance
* Average Distance
* Monthly Analysis
* Quarterly Analysis
* Revenue by Vehicle Type
* Top Pickup Locations
* Top Drop Locations
* Average Rider Ratings
* Average Driver Ratings

### Vehicle Page

* Vehicle-wise Booking Count
* Vehicle-wise Revenue
* Revenue Contribution %
* Vehicle Performance Trends

### Revenue Page

* Revenue by Customer
* Revenue by Vehicle
* Revenue by Payment Method
* Monthly Revenue Analysis
* Quarterly Revenue Analysis

### Rider Page

* Ride Cancellation Analysis
* Cancellation by Payment Method
* First Riders
* Return Riders
* Regular Riders

## Business Requirements Implemented

* KPI Tracking
* Vehicle Performance Analysis
* Revenue Analysis
* Rider Segmentation
* Cancellation Analysis
* Monthly and Quarterly Trends
